Lisää oma sähköpostiosoitteesi lomake.php:

$email_to = ""; // lisää tähän oma osoiteesti

Kopioi kaikki em. tiedostot samaan hakemistoon palvelimellesi (esim. local tai public)

Varmista, että kyseisellä palvelimella on php-käytössä: tarvittaessa käynnistä esim. XAMPP.

Kyseinen lomake ei lähetä tietoja sähköpoistiin - toimii ainoastaan tiedon katselun mallintamiseen.

Lomakkeen tietoturva on tarkoituksella heikko - lomaketta ei tule käyttää sellaisenaan minkään tiedon välittämiseen / keräämiseen tuotantokäytössä.
